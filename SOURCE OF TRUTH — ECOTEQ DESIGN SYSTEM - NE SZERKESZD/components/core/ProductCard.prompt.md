# ProductCard

Machine / product card — photo under the navy wash with a category Tag, title, short copy and a green read-more. The core building block of the product grid. Composes `Tag`.

```jsx
<ProductCard
  image="/img/shredder.jpg"
  category="Ipari darálás"
  title="ECO-Shred 400"
  description="Nagy áteresztőképességű ipari daráló kemény hulladékhoz."
  cta="Részletek"
  onClick={…}
/>
```
