# IconButton

Square icon-only button — rail toggles, close, pagination. Always pass `label` for a11y; icon goes in children.

```jsx
<IconButton label="Bezárás"><X/></IconButton>
<IconButton variant="solid" pill label="Tovább"><ArrowRight/></IconButton>
<IconButton variant="outline-light" label="Menü"><Menu/></IconButton>  {/* on navy */}
```
