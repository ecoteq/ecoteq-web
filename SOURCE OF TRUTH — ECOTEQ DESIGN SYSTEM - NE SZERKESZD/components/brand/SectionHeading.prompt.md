# SectionHeading

The standard ECOTEQ section opener: green overline + ExtraBold heading + optional lead. Composes `Overline`.

```jsx
<SectionHeading
  overline="Megoldások"
  title="Ipari hulladékból érték"
  lead="A darálástól a brikettálásig — tervezhető, üzemeltethető, szervizelhető rendszerek."
/>
<SectionHeading tone="dark" align="center" title="Kapcsolat" />
```

- `tone="dark"` for navy / photo backgrounds.
- `align="center"` to center the block.
