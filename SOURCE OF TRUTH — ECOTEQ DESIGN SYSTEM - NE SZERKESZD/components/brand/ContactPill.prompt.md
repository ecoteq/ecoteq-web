# ContactPill

The signature outlined contact chip from the navy brand rail — green icon + text, fills green on hover. Use for email/phone.

```jsx
<ContactPill icon={<Mail/>} href="mailto:info@ecoteq.hu">info@ecoteq.hu</ContactPill>
<ContactPill icon={<Phone/>} tone="light" href="tel:+36...">+36 00 000 000</ContactPill>
```
